# ShopEasy Git Project

## Overview

This project demonstrates basic Git operations including branching, committing, and merging, as part of a learning assignment.

## File Description

* `spec.txt` : Contains the ShopEasy cart specifications and sample content for testing Git operations.

## Git Workflow Demonstrated

1. Initialized a Git repository.
2. Created and linked to a remote repository.
3. Created branches (A and B) and edited files in each.
4. Committed changes with meaningful messages.
5. Merged branches into main.
6. Resolved a merge conflict (simulated by editing the same line in different branches).

## Key Commands Used

* `git init` : Initialize repository
* `git remote add origin <URL>` : Link to remote
* `git add <file>` : Stage file for commit
* `git commit -m "message"` : Commit staged files
* `git branch -b <branch>` : Create new branch
* `git checkout <branch>` : Switch branches
* `git merge <branch>` : Merge branch into current
